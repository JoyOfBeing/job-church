### SLIDE 12 — THE STRUCTURE

## Regenerative Community Organism (RCO)

JOB is structured as an RCO — a model designed for purpose-driven organizations that create more life than they consume.

```
┌─────────────────────────────────────────────────────┐
│                  SHARED PURPOSE                      │
│    "Resource humans to be fully human"               │
│                                                      │
│  ┌────────────────────┐  ┌────────────────────────┐ │
│  │  JOB CHURCH        │  │  B3.0 HOLDING CO       │ │
│  │  (Non-Profit       │  │  (Limited Company)     │ │
│  │   Association)     │  │                        │ │
│  │                    │  │  • Consulting           │ │
│  │  • Holds purpose   │  │  • Cohorts              │ │
│  │  • Community       │  │  • Franchise            │ │
│  │    governs it      │  │  • Talent placement     │ │
│  │  • 10% preferred   │  │                        │ │
│  │    shares          │  │  Revenue grows the      │ │
│  │  • 50% board       │  │  business, then flows   │ │
│  │    appointments    │  │  back to the commons    │ │
│  │  • Veto on purpose │  │                        │ │
│  │    changes         │  │                        │ │
│  └────────────────────┘  └────────────────────────┘ │
└─────────────────────────────────────────────────────┘
```

**Why this structure:**
- Investors get real returns from the for-profit side
- Purpose can never be corrupted — the community holds veto power
- As B3.0 matures, excess resources flow back into the commons
- The organization has a natural lifecycle — it can seed new organizations, not just extract value
- Modeled after Patagonia's purpose trust, but purpose-native from day one
